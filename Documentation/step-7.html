Step seven

Deploy project to firebase hosting

In this step i will show you how to deploy your project to firebase hosting. First of all we have to need run a command

npm install -g firebase-tools

If this command has been run before, then there is no need to run it

When finished firebase-tools installed then run this command

firebase login

The account with which you created the firebase project must be logged in

When login successfully then run this command

firebase init

Press Y then enter

Use an existing project then press enter

Wait for firebase initialization completed

When firebase initialization completed then run a command for build our vuejs project

npm run build

Wait few minutes for build our project

When project builds successfully then run this command for deploy your project to firebase hosting

firebase deploy

When firebase deploy successfully then Go to Firebase console and check Hosting Dashboard. Firebase provide 2 default domain for check your project real server. You can also add your custom domain
