Step four

Setup firebase Storage

At this stage we will add some images manually. Firstly go firebase console and go to you project then Get started Firebase Storage

Simply click on Next button

Set the cloud storage location

Then upload some images. Go to codecanyon downloaded unZip folder then go Uploads folder here is some images for application. You can change the images if you want, but the image name and image extensions must be the same. For example: the logo of the application here is logo.svg If you want to change it, rename your logo as logo.svg

Upload all images in the Uploads folder

Then change the firebase Storage Rules. Simply allow read and write
