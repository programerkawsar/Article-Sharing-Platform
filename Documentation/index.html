Step one

Firebase installation

First go to this link --> Firebase then click on the Go to console button. You must have a Gmail account logged in and if not, login fast.

Then click on the Add project button and create a new project

Write your Project name and edit Project ID if you want but not necessary then click on Continue button

Set Default Account for Firebase then click on Create Project button

Wait few seconds for building project

When building successfully then click on Continue button

Firebase provide lots of features we are using some pacific feature for our application. Follow the red spots shown in the screenshot

Write your App nickname and also checked Firebase Hosting then select your App ID then click on Register app button

Just avoid the section click on Next button

Copy the npm command and click on Next button

I will use these commands in the next step. Avoid them now and click on Continue to console button

Follow the red spots shown in the screenshot

Click on Setting icon

Scroll down and copy the codes


Step two

Vue CLI installation

First read the Vue CLI installation article --> Vue CLI and follow the all in installation process. After reading Vue CLI installation article then create a creating a project. This article will help you to create a project --> Vue Create. If you don't understand vue official documentation don't worry i will show step by step. follow the step below

First of all let me say I use NPM package manager. If you don't know about NPM, you can check it out here --> NPM package manager
First run the command on CMD

npm install -g @vue/cli

After finishing installing vue cli then run the command

vue create your-project-name

Manually select features

Select Babel, Router, Vuex

Press Y for use history mode

Select package.json

Press N

Wait few seconds for finishing installation

After finishing installing then go to project directory

Then open project in code studio


Step three

Initialize Project

I am using vscode for initialize project because it's very easy and developer friendly. Ok let's start step three. First unZip the codecanyon downloaded file then open ArticleShare folder and...

  • Go to Files folder and copy all files
  • Go to your installed vue project folder then paste all files and replace all files

Open Code Editor and open 2 files package.json, package-lock.json then change package name and replace it your project name

Run this command for install all of necessary plugin for your application

npm install

Wait few seconds to install all plugins

Go firebase console then go to Your project > Project settings scroll down find Firebase SDK snippet section and checked config the copy the firebaseConfig codes

After install all of plugin then Go to project folder -> src -> firebse -> init.js and paste firebaseConfig codes

If you want to check your project on development server to run this command

npm run serve


Step four

Setup firebase Storage

At this stage we will add some images manually. Firstly go firebase console and go to you project then Get started Firebase Storage

Simply click on Next button

Set the cloud storage location

Then upload some images. Go to codecanyon downloaded unZip folder then go Uploads folder here is some images for application. You can change the images if you want, but the image name and image extensions must be the same. For example: the logo of the application here is logo.svg If you want to change it, rename your logo as logo.svg

Upload the images in the Uploads folder

Then change the firebase Storage Rules. Simply allow read and write


Step five

Setup the application

Enable firebase authentication sign-in providers. First we will enable email/password provider. You also enable social login like Google, Facebook, Twitter but you have to need create social media app.

Just enable email/password provider then click on Save button

We will use firestore for store your data. Click on Create database button

We will first launch the application in test mode to test it

Then select your cloud firestore location

We have need to change firestore database rules

allow read, write : if request.auth != true; allow create, delete, update : if request.auth != null && exists(/databases/$(database)/documents/users/$(request.auth.uid));
Add two line and save or finished it

We have brought a lot of data in our application in ascending and descending order so you need to add index to firestore database.

The following screenshot lists all the indexes. Add all the indexes to your database by looking at the screenshot

For example, I am adding the first index of the screenshot.

The index will take some time to build. You add all the indexes shown in the screenshot one by one

Start the development server and open browser press the localhost running url like http://localhost:8080

npm run serve

Go to register page

Register a account for admin

When registration successful then go to firestore database and go users collection > document > scroll down and find role field then set value 1

Now admin account has been activated. Go to base url and look on dropdown navigation added a page for admin panel

Now you can control everything from admin panel

Set metadata manually

The application index.html meta data not dynamic so we will add some info manually. You set the data as you like and add your favicon. This information will show when the website is not fully loaded and also show social sharing page. The data will change after the website is loaded.
